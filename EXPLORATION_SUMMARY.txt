=============================================================================
MAIN EXECUTION FLOW
=============================================================================

1. User sends query from HoudiniMindPanel UI
   ↓
2. AgentLoop.run() processes query
   - Detects intent (BUILD/VEX/HDA/DEBUG/RESEARCH)
   - Validates scene references
   - Optionally generates plan (adaptive)
   - Validates preconditions
   ↓
3. Tool Selection Loop (max 16 rounds)
   - LLM selects next tool
   - Arguments validated against JSON Schema
   - Tool executed with 90s timeout
   - Scene state captured and analyzed
   - Critic gates dangerous operations
   ↓
4. Observation & Verification
   - Screenshot viewport (vision-enabled)
   - Analyze geometry changes
   - Semantic scoring (optional)
   - Completion detection
   ↓
5. Response Generation
   - Compile observations
   - Stream to UI in real-time
   - Log to session.md file
